알라딘 API 를 활용한 도서 관리 시스템 - 1

1. API 를 통해서 데이터 가져오기
  - Bestseller 기준으로 요청

2. DB 에 필요한 데이터만 파싱하여 저장하기
  - 도서번호(isbn)
  - 제목(title)
  - 내용(description)
  - 출간연도(pubDate)
  - 작가(author)
  - 판매가격(priceSales)

  - 단, 저장 시 중복된 데이터는 저장하지 않도록 코드 구성

3. 전체 데이터 조회하기
  - 제목, 내용, 작가, 가격을 출력
  - 단, 제목은 단어가 5개 초과 시 ... 으로 생략 (DTL filter 활용)
  - 제목 클릭 시 상세 페이지로 이동
  - [도전] 최대 금액 입력 -> 금액 이하인 책들만 리스트로 출력
    - 힌트: 금액 입력 후 URL 은 다음과 같이 구성할 수 있다.(http://127.0.0.1:8000/books/?max_price=10000)

4. 상세 데이터 조회하기
  - 전체 데이터를 모두 출력
